# Installation

## Download

Download the Android SDK AAR Library from [Here](https://github.com/zach-luna/SDK-Dist/blob/main/LunaSDK.aar)

## Setup

In order to use the sdk you need to import the library following these steps:

1. Copy .aar to libs/ Folder

```
your-app/
├── libs/
│   └── LunaSDK.aar
```

2. Update build.gradle in the module (usually app/build.gradle):

```
repositories {
    flatDir {
        dirs 'libs'
    }
}

dependencies {
    implementation(name: 'LunaSDK', ext: 'aar')
}
```

3. Add required dependencies in the module (usually app/build.gradle) for LunaSDK to work:

```
dependencies {
    implementation("com.squareup.retrofit2:retrofit:2.11.0")
    implementation("com.squareup.retrofit2:converter-moshi:2.11.0")
    ksp("com.squareup.moshi:moshi-kotlin-codegen:1.15.0")
    implementation("com.squareup.moshi:moshi-kotlin:1.15.0")
}
```

4. In your module app/build.gradle also add these plugins:

```
plugins {
    id("kotlin-kapt")
    kotlin("kapt")
    id("com.google.devtools.ksp")
}
```

5. rebuild your project
